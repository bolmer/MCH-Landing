# MCH Landing Page - Product Section Specifications

Este documento detalla la estructura y el contenido de la sección de productos de la Landing Page de MCH.

## 1. Encabezado de Productos
- **Clase CSS:** `mx-auto max-w-[1400px] px-6 py-24`
- **Texto:** 
  > PRODUCTS
  > A suite of tools that get things done.
  > Each product targets a specific data problem. Deployed independently and scaled with operations.
- **Detalle:** La frase "get things done" utiliza la clase `font-serif not-italic font-medium text-crust-golden`.

## 2. Grid de Productos / MVP
- **Clase CSS:** `mt-16 pb-32`
- **Tecnologías:** LOCAL LLMS · RAG · REDSHIFT · FASTAPI
- **Producto:** MVP - Data Governance Chat Bot
- **Descripción:** 
  > Instant queries about policies and metadata.
  > Conversational assistant specialized.
